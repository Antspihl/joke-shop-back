## Features

- Browse a collection of various setups.
- Purchase punchlines to the selected setups.
- Add selected punchlines to a cart for easy checkout.
- Convenient payment options for a seamless shopping experience.

## Technologies Used
 - Java 18
 - Spring Boot
 - Spring Security
 - Spring Data JPA
 - Hibernate
 - PostgreSQL
 - Gradle
 - Lombok
 - MapStruct
